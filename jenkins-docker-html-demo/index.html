🚀 Hello from Jenkins + Docker CI/CD

Version 1.0

This website is automatically deployed using a modern CI/CD pipeline. Every time a developer pushes changes to GitHub, Jenkins automatically detects the commit, builds a new Docker image, stops the old container, and deploys the updated version instantly!

🔄 Automated Pipeline Workflow:

  1. Developer pushes code to GitHub
  2. Jenkins detects the new commit
  3. Jenkins checks out the latest code
  4. Jenkins builds a new Docker image
  5. Jenkins stops and removes the old container
  6. Jenkins starts a new container with the updated code
  7. Changes are live immediately! 🎉
